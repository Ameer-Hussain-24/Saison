# 课程周数选择器 - 用户指南

## 功能简介

课程周数选择器允许你精确控制课程在哪些周上课，支持：
- 🎯 **自定义周数** - 选择任意周数组合（如第1,3,5,7周）
- ⚡ **快捷模式** - 一键选择全周、单周或双周
- 📅 **日期模式** - 通过日期范围选择周数
- 📊 **实时统计** - 显示已选择的周数

## 使用方法

### 1. 添加课程时选择周数

#### 步骤1：打开添加课程界面
- 在课程表页面点击右下角的 **"+"** 按钮

#### 步骤2：填写基本信息
- 输入课程名称（如"高等数学"）
- 选择星期（如"周一"）
- 设置上课时间

#### 步骤3：选择周数
1. 找到 **"周模式"** 部分
2. 点击右侧的 **"查看详情"** 按钮
3. 在弹出的对话框中选择周数

#### 步骤4：保存课程
- 点击 **"添加课程"** 按钮完成

### 2. 使用周数选择器

#### 方式1：快捷选择（推荐）
在对话框顶部有三个快捷按钮：
- **全周** - 选择所有周（1-18周）
- **单周** - 选择奇数周（1,3,5,7...）
- **双周** - 选择偶数周（2,4,6,8...）

点击任一按钮即可快速选择。

#### 方式2：手动选择
1. 在网格中点击具体的周数按钮
2. 再次点击可取消选择
3. 可以选择任意组合

**提示：** 手动选择后，快捷模式会自动取消。

#### 方式3：日期模式
1. 点击顶部的 **"日期模式"** 标签
2. 选择开始日期和结束日期
3. 选择重复模式（每周/单周/双周）

### 3. 查看选择结果

#### 在对话框中
- 底部显示 **"已选择 X 周"**
- 选中的周数按钮会高亮显示

#### 在课程卡片上
- 显示周数标签：
  - "每周" - 全周模式
  - "单周" - 单周模式
  - "双周" - 双周模式
  - "自定义" - 自定义周数

### 4. 编辑课程周数

1. 点击课程卡片进入编辑界面
2. 点击 **"查看详情"** 按钮
3. 修改周数选择
4. 点击 **"确定"** 保存
5. 点击 **"保存修改"** 完成

## 使用场景

### 场景1：每周都上的课程
**示例：** 英语课每周都上

**操作：**
1. 点击"查看详情"
2. 点击 **"全周"** 按钮
3. 点击"确定"

**结果：** 课程在所有周都会显示

### 场景2：单双周课程
**示例：** 体育课单周上

**操作：**
1. 点击"查看详情"
2. 点击 **"单周"** 按钮
3. 点击"确定"

**结果：** 课程只在第1,3,5,7...周显示

### 场景3：特定周数的课程
**示例：** 实验课在第2,4,6,8,10周上

**操作：**
1. 点击"查看详情"
2. 手动点击第2,4,6,8,10周的按钮
3. 点击"确定"

**结果：** 课程只在指定的周显示

### 场景4：前半学期的课程
**示例：** 某课程只在前9周上

**操作：**
1. 点击"查看详情"
2. 手动点击第1-9周的按钮
3. 或使用"全周"后取消第10-18周
4. 点击"确定"

**结果：** 课程只在前9周显示

## 常见问题

### Q1：如何取消已选择的周数？
**A：** 再次点击该周数按钮即可取消选择。

### Q2：可以选择不连续的周数吗？
**A：** 可以！你可以选择任意组合，如第1,3,7,15周。

### Q3：快捷模式和手动选择可以混用吗？
**A：** 可以先用快捷模式，然后手动调整。但手动选择后快捷模式会取消。

### Q4：如何查看课程在哪些周上课？
**A：** 
1. 点击课程卡片进入编辑
2. 点击"查看详情"
3. 查看高亮显示的周数

### Q5：最多可以选择多少周？
**A：** 取决于学期设置，默认是18周，可以全部选择。

### Q6：如何修改学期总周数？
**A：** 在设置页面的"课程设置"中可以修改学期总周数（16-24周）。

## 技巧和建议

### 技巧1：使用快捷模式提高效率
对于规律的课程（全周、单周、双周），优先使用快捷按钮。

### 技巧2：先选后调
可以先用快捷模式选择大部分周数，然后手动调整个别周。

### 技巧3：查看统计信息
底部的"已选择 X 周"可以帮助你确认选择是否正确。

### 技巧4：合理规划
在学期开始时就规划好哪些课程在哪些周上，可以避免后期频繁修改。

## 界面说明

### 周数按钮
- **圆形按钮** - 每个代表一周
- **蓝色填充** - 已选择
- **灰色轮廓** - 未选择
- **数字** - 周数（1-18）

### 快捷按钮
- **全周** - 选择所有周
- **单周** - 选择奇数周
- **双周** - 选择偶数周
- **蓝色背景** - 当前激活的模式

### 统计信息
- 显示已选择的周数
- 实时更新
- 蓝色文字

### 操作按钮
- **取消** - 放弃修改，关闭对话框
- **确定** - 保存选择，关闭对话框
- **确定按钮禁用** - 至少需要选择一周

## 注意事项

1. **至少选择一周** - 必须至少选择一周才能保存
2. **周数范围** - 只能选择1到学期总周数之间的周
3. **手动选择优先** - 手动选择后快捷模式会自动取消
4. **保存后生效** - 必须点击"确定"和"添加课程"才会保存
